محدودیت‌ها:

- حداکثر ماتریس 3×3
- حداکثر بردار 3 تایی
- ضرب خارجی فقط برای بردار 3 تایی کار می‌کنه
- دترمینان و معکوس فقط برای ماتریس 2×2 و 3×3



مثال‌های استفاده:

مثال 1: جمع دو ماتریس 2×2
ورودی:
  ماتریس 1:  [1  2]
              [3  4]
  
  ماتریس 2:  [5  6]
              [7  8]

نتیجه:       [6   8]
              [10  12]


مثال 2: ضرب داخلی
ورودی:
  بردار 1: <1, 2, 3>
  بردار 2: <4, 5, 6>

نتیجه: 32
چون: (1×4) + (2×5) + (3×6) = 4 + 10 + 18 = 32


مثال 3: اندازه بردار
ورودی:
  بردار: <3, 4>

نتیجه: 5
چون: √(3² + 4²) = √(9 + 16) = √25 = 5


خطاهایی که مدیریت میشن:
 وقتی ابعاد ماتریس‌ها برای جمع یا تفریق مناسب نباشه
 وقتی ابعاد ماتریس‌ها برای ضرب مناسب نباشه
 وقتی بخواییم دترمینان ماتریس غیر مربعی رو حساب کنیم
 وقتی دترمینان صفر باشه و نشه معکوس حساب کرد
 وقتی بخواییم ضرب خارجی بردار 2 تایی رو حساب کنیم
 وقتی بخواییم بردار صفر رو نرمال کنیم
